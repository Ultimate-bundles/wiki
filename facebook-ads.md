# Facebook Ads

## Naming convention
Campaign level = "Bundle code [kind of  campaign]".  E.g. UHOMB [Purchase Campaign]
Ad set level = "Full length description (sub description)".  E.g. Retargeting, Interest (cats)
Ad level = "Copy 1 - Image 1"
